- 객체에 메세지 보내기

### 리팩토링(2) - 객체의 책임과 응집도
- IO 통합 적용
- 일급 컬렉션 적용

- Order 객체

### 리팩토링(3) - 관점의 차이로 달라지는 추상화
- FileHandler 를 바라보는 관점

- 객체의 책임과 응집도 - IO 통합, 일급 컬렉션, display(), Order 추출
- 추상화 관점의 차이 - FileHandler
