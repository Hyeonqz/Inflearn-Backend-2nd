### [참여 계기]
Test 에 대해서 고민을 자주 하였고, 테스트 강의를 들어볼까 하던 차에 <br>
감사하게도 인프런에서 위 이벤트가 있어 겸사 겸사 참여를 하였음 <br>
추가적으로 나와 비슷한 고민을 하는 주니어 개발자 분들과 직접 소통을 하며 인사이트를 얻어 보는걸 목표로함
